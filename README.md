# igame-main-guide

## 简介

igame-main-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，主要用于存放各类静态页面资源，方便索引、预览与版本管理。

## 目录说明

```
igame-main-guide/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 公共资源文件（如图片、样式、脚本等）
├── index.html      # 仓库根目录的索引页面（可选）
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件对应一个独立页面，命名清晰，便于查找。
- **assets/**：存放页面共用的资源，避免重复引用。
- **index.html**：如存在，可作为页面列表或导航入口。

## 页面归档说明

仓库中的 HTML 页面均为独立归档，各自包含完整的结构、样式与逻辑。  
页面之间无强依赖关系，可单独部署或预览。

归档内容可能包括：
- 工具页面
- 信息展示页
- 实验性页面
- 其他静态功能页面

所有页面均为纯前端实现，不依赖后端服务。

## 维护说明

- 新增页面请放入 `pages/` 目录，并确保命名不与已有文件冲突。
- 如有公共资源更新，请同步修改 `assets/` 下对应文件。
- 建议每次提交附带简要说明，方便追溯变更。
- 本仓库不主动推广或指向外部链接，仅作存档与分发用途。

## 贡献

欢迎提交 Issue 或 Pull Request 来改进页面质量、修复问题或增加归档内容。  
请确保新增内容符合仓库定位，且不引入外部依赖或敏感信息。
